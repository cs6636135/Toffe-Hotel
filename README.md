##overview
===========
Toffe Hotel is a hotel booking system analysis project that focuses on requirement engineering and system modeling. 
The project translates stakeholder needs into a structured SRS supported by UML diagrams and UI prototypes.
==========
#Objectives
-Analyze user requirements for a hotel booking system
-Define system requirements (functional and non-functional)
-Design system workflows and high-level data model
-Produce Software Requirements Specification (SRS) documentation
-Create a UI prototype for system visualization

#Deliverables
-Software Requirements Specification (SRS)
-UML Diagrams:
 Use Case Diagram
 Class Diagram
 Activity Diagram (Swimlane) 
 ER Diagram
-Figma Prototype
